# Введение в систему контроля версий Git

**Git** - система управления версиями;

**GitHub** - онлайн-платформа, в которую внедрена VCS Git;

**VCS(Version Control System)** - система контроля версий;

**Git-репозиторий** - папка, отвечающая за работу VCS в рабочем проекте(папка с проектом, в которой проинициализрован git-репозиторий);

**Виды репозиториев**:
- **Локальный репозиторий** - репозиторий, который размещен на локальном устройстве(компьютер, ноутбук и т.п);
- **Удаленный репозиторий** - репозиторий, который размещен на сервере.
