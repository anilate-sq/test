# Список основных комманд

## Подключение
- **git remote add <переменная> <ссылка>** - создание подключения к удаленному репозитория;
- **git remote -v** - просмотр списка подключений;
- **git remote remove <ссылка>/<переменная>** - удаление подключения;
- **git clone** - полная выгрузка удаленного репозитория в локальный.

## Работа с изменениями
- **git add <файл>** - добавление изменений в область подготовленных изменений перед коммитом;
- **git commit -m <комментарий>** - создание коммита с комментарием об внесенных изменениях;
- **git push <ссылка> <ветка>** - отправка внесенных изменений по указанному адресу, в указанную ветку;
- **git pull <ссылка> <ветка>** - выгрузка данных с указанного адреса, указанной ветки в свой репозиторий с автослиянием;
- **git fetch <ссылка> <ветка>** - тоже самое, что и git pull, но без автослияния;
- **git restore <файл>** - отмена внесенных изменений.

## Работа с историей
- **git log** - просмотр истории изменений;
- **git diff** - сравение коммитов, файлов и веток;
- **git show <коммит>** - просмотреть информацию о коммите.
